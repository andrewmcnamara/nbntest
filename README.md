# NBN Test
Test your nbn connection

This program can be used to record your modem statistics and the results of speedtest.net tests over a period of time

# Supported Outputs

* InfluxDB
* Console

# Supported Modems

* TPLink TD-9970
* Generic Telnet via LUA
